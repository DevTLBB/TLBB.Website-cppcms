# CppCMS.Project

# Installation:


```bash
sudo apt-get install build-essential libpcre3-dev libicu-dev libgcrypt11-dev zlib1g-dev cmake

git clone git@github.com:HadesD/CppCMS.Project.git
cd CppCMS.Project
git submodule update --init --recursive
cd build
cmake ../cppcms
make
sudo make install
cd ..
./server.sh
```
